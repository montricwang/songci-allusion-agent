# Skill: Wang Guowei Ci Analysis

## Goal
依据王国维《人间词话》的核心理论，对输入诗词进行结构化赏析。

本 skill 不做泛泛文学评论，而是根据以下几个核心概念进行判断：

- 境界
- 有我之境 / 无我之境
- 写境 / 造境
- 隔 / 不隔
- 真景物 / 真感情

输出必须结构化，方便程序解析。

---

## Core Theory (from 人间词话)

1. **词以境界为最上。**

2. **有有我之境，有无我之境。**
   - 有我之境：以我观物，物皆著我之色彩。
   - 无我之境：以物观物，不知何者为我何者为物。

3. **有造境，有写境。**
   - 写境：来自真实经验的景象
   - 造境：作者构想或理想化的境界

4. **隔 / 不隔**
   - 不隔：语言与意象能直接抵达情感
   - 隔：表达空泛、堆砌辞藻、感受被遮蔽

5. **真**
   - 真景物
   - 真感情

---

## Input

用户提供：

- title: 作品标题（可选）
- author: 作者（可选）
- text: 诗词正文（必填）

示例：
- title: 浣溪沙
- author: 晏殊
- text: 一曲新词酒一杯，去年天气旧亭台。

## Output Fields

必须输出以下字段：

- has_jingjie  
  是否具有“境界”

- realm_type  
  境界类型  
  可选值：
  - 有我之境
  - 无我之境
  - 难判定

- creation_mode  
  造境 / 写境  
  可选值：
  - 写境
  - 造境
  - 兼有
  - 难判定

- transparency  
  “不隔”的程度  
  0.0 - 1.0

- sincerity  
  真情真景程度  
  0.0 - 1.0

- explanation  
  100-200字简要分析

---

## Analysis Rules

1. **优先判断是否有境界。**
2. 判断物我关系 → 有我 / 无我。
3. 判断景象来源 → 写境 / 造境。
4. 判断表达是否“隔”。
5. 判断是否具有真情真景。
6. 若证据不足，可以输出“难判定”。

---

## Style Constraints

- 避免空泛赞美
- 分析要克制简洁
- 尽量引用原句中的意象作为证据
- 不编造历史背景
